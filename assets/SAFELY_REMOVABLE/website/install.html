Installation

Installing this theme is simple:

First, download the theme's contents, which can be done by clicking the button below (it links directly to the Download ZIP link in the github page).

Next, extract the contents of the .zip file to your themes folder: refer to here for more information.

Finally, you may remove the SAFELY_REMOVABLE folder that's in the assets folder, as it only provides screenshots and other resources not required to run the theme.
